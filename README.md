# BC-AL-Learning
AL Programming Practice – Daily Learning Journey
Welcome to my AL (Application Language) practice repository, where I document my daily learning, exercises, and experiments while developing extensions for Microsoft Dynamics 365 Business Central.

This repository serves as a personal learning log, showcasing my progress as I improve my skills in:

-AL syntax and structure
-Tables, Pages, Codeunits
-Extensions development
-Triggers & Events
-Reports & Queries
-API pages
-Integrations
-Practice tasks and real-world scenarios
